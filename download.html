Install in under a minute

Three steps, zero friction.

The extension isn't in the Chrome Web Store yet (review pending). Load the unpacked build — same thing, takes 30 seconds.

1

Download & unzip

Click the download button above. You'll get passwordn-extension.zip. Unzip it anywhere on disk.

Download .zip
2

Open chrome://extensions

Toggle Developer mode in the top right. Click Load unpacked and select the folder you unzipped.

3

Create your vault

Click the Passwordn icon in the toolbar, set a master password, and you're in. Everything syncs with the web app.

Open web vault →
Inline autofill

A little badge. A big time-saver.

Passwordn quietly attaches an Autofill badge to password fields on any site. Click it, pick an account, you're signed in.

  • ⌘⇧L Fill without even opening the popup.
  • Strict Domain-locked — never fills into a lookalike.
  • Local Matching happens in your browser, not on a server.
